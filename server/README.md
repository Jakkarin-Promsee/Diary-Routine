**Game play**
- hellicopter เคลื่อนที่ไปด้านหน้าเรื่อยๆ
- กด enter เพื่อปล่อยน้ำ
- ไม่ได้ดับไฟป่า -> เสียเเต้ม
- โดนระเบิด -> game over
- โดนบ้านคน <10 -> ได้รับเเต้ม

- ถ้าเเต้มมากกว่ากว่า 10 ->win
