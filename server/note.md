# HTTP METHOD

- GET: Retrieve Data only

  - Can access from both url and fetch

- POST: Create New Data

  - Can acess from only fetch

- PUT: Update whole data

  - Can acess from only fetch

- PATCH: Update some part of data

  - Can acess from only fetch

- DELETE: Delete all Data

  - Can acess from only fetch
